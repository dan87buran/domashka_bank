Проект представляет собой набор инструментов для обработки банковских операций, включая фильтрацию и сортировку данных.

## Новая функциональность

Добавлена поддержка чтения транзакций из различных форматов файлов:

- read_csv_transactions(file_path: str) -> List[Dict] - чтение из CSV
- read_excel_transactions(file_path: str) -> List[Dict] - чтение из Excel
