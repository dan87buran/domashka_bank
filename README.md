# domashka_bank
# Виджет банковских операций

## Описание проекта
Проект представляет собой набор инструментов для обработки банковских операций, включая фильтрацию и сортировку данных.

## Установка

1. Клонирование репозитория:
```bash
git clone <URL_репозитория>
cd виджет-банковских-операций
```
2.Написание тестов:
```bash
Все функции покрыты тестами

Используются параметризация и фикстуры

Покрытие составляет более 80%

Линтеры выдают не более 4 ошибок

При запуске тестов нет падений
```
